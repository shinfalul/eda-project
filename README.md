Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a digital advertising campaign dataset.
The dataset contains campaign performance data from multiple online advertising platforms such as Google Ads, Meta Ads (Facebook/Instagram), and TikTok Ads.

The goal of this project is to analyze campaign performance, understand relationships between marketing metrics, and generate insights that can help improve advertising strategies.

🎯 Objectives

The main objectives of this project are:

Understand the structure and quality of the dataset

Perform data cleaning and preprocessing

Conduct univariate analysis to understand individual variables

Perform bivariate analysis to identify relationships between variables

Use statistical analysis and correlation to understand key performance drivers

Generate insights and recommendations for marketing optimization

Dataset Description

The dataset includes the following key metrics:

Impressions → Number of times an ad was displayed

Clicks → Number of times users clicked the ad

CTR (Click Through Rate) → Clicks divided by impressions

CPC (Cost Per Click) → Cost for each click

Ad Spend → Total money spent on ads

Conversions → Number of desired actions (purchase/signup)

CPA (Cost Per Acquisition) → Cost per conversion

Revenue → Revenue generated from ads

ROAS (Return on Ad Spend) → Revenue divided by ad spend

📈 Key Insights

Higher ad spend generally increases impressions and conversions

Clicks strongly influence conversions

A small number of campaigns generate a large portion of conversions

Platform performance varies, with some platforms generating higher conversions

ROAS depends more on campaign efficiency than budget size

💡 Recommendations

Based on the analysis:

Focus budget on high-performing campaigns

Analyze successful campaigns to replicate strategies

Optimize low-performing campaigns

Improve targeting and creative strategies

Continuously monitor campaign performance
